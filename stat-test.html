Formula rules stored on page

Observed Stats Mode: - Use current in-game black card HP/ATK directly. - Do NOT add Fellowship again. It is already baked into the observed account stats. - Add equipment separately for power only. Formula Rebuild Mode: RawStat = RefStat200 / 294 × (Level + 10) × MLB AccumStat = RawStat + AccountFlatFellowship + BoostStat AwakenedStat = AccumStat × AwakeningMultiplier FinalBlackCardStat = round(AwakenedStat × (1 + PotentialPct / 100)) + AscensionStat Power: CharacterBasePower = FinalBlackCardHP × 12 + FinalBlackCardATK × 60 MasteryPower = CharacterBasePower × 0.0025 × MasteryLevel EquipmentPower = WeaponHP×6 + WeaponATK×30 + AccessoryHP×6 + AccessoryATK×30 + AccessorySPD×180 UnitPower = round(CharacterBasePower + MasteryPower + EquipmentPower) Account Rank BP is battalion/profile only. It is never added to an individual character card power.

Advanced modifiers

Account fellowship is account-specific. Defaults are your current profile: Flat HP +4900, Flat ATK +750, % HP 0, % ATK 0. These apply only in Rebuild Mode unless explicitly forced.
